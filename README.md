# GoLang REST API

REST API W / GoLang and MongoDB

## Description

Simple REST API over CRUD Operations through MongoDB Atlas

## Getting Started

### Dependencies
	github.com/gofiber/fiber v1.14.6
	github.com/mitchellh/mapstructure v1.4.1
	go.mongodb.org/mongo-driver v1.7.1



### Executing program

For Unix Users
```
$ chmod +x hello.go
$ ./hello.go
```

For Unix Users
```
$ go run ./main.go
```
